(1) Data was collected by Zachary (1977). The data as presented here was taken from the R package igraphdata (Csardi, 2015). Information on node attributes was added from original study.

Sources:

Zachary, W.W. (1977). An Information Flow Model for Conflict and Fission in Small Groups. Journal of Anthropological Research 33(4), 452-473.
